# Vault Root — Index

The long-term memory for **the-five-agents** — a content-creation agent team project. Each folder holds topic-based Markdown notes following the Obsidian Vault Workflow.

## Folders

- [[Meeting Notes/_index|Meeting Notes]] — code, architecture, decisions, file inventories, session logs
- (Content Briefs, Publishing Log, Brand Guidelines will be created when first needed)

## Conventions

- **One `.md` per topic**, lowercase-hyphenated filename, no date in filename
- Each topic file: `# Title` → `## Overview` → `## Open Questions` → `## Session Log`
- Newer session entries appended at the **bottom** of the Session Log
- Cross-links use `[[wikilinks]]`, never markdown links
