# cavani/reference/ Approved Style Samples

תיקייה זו מכילה מאמרים שאושרו בעבר על ידי valverde ומשמשים כעוגן סגנוני ל-cavani. הסוכן עושה `Glob cavani/reference/*` ואז `Read` על כל קובץ כחלק מה-Workflow שלו, חולץ דפוסי טון/מבנה/קצב, ויישם אותם על המאמר החדש.

## איך cavani משתמש בקבצים האלה

1. **Glob** מאתר את כל ה-samples
2. **Read** טוען את התוכן (כולל header HTML המסביר את ההקשר)
3. cavani מזהה: tone, structure, sentence rhythm, vocabulary, cultural register
4. מיישם דפוסים רלוונטיים על המאמר החדש שהוא כותב

## כללים להוספת sample חדש

1. הקובץ חייב להיות **מאמר שעבר QA אצל valverde** (✅ APPROVED, סבב סופי).
2. הוסף **HTML header בראש הקובץ** שמתאר: type, domain, quality, use case, notes for cavani.
3. שמור על קונבנציית שמות: `<YYYY-MM-DD>-<slug>-<lang>-sample.md`.
4. עדכן את הרשימה בסוף ה-README הזה.
5. שלח את ה-sample ל-valverde לבדיקה משנית, שאין סתירה עם ה-style-guide.

## דומיינים לא מכוסים

נכון לעכשיו ה-samples ממוקדים בעיתונות ספורט. דומיינים אחרים (עסקי, טכני, שיווקי, תרבות) **אין להם רפרנס מאומת**. אם cavani מקבל בקשה בדומיין כזה, עליו לדווח ל-Enzo ולא לאלתר.

## קישור ל-style-guide

הסטייל-גייד הקנוני יושב ב-`cavani/style-guide.md`. הרפרנסים האלה הם **הדגמה** של הכללים בפועל, לא תחליף לכללים. אם יש מתח בין הסטייל-גייד לבין רפרנס, הסטייל-גייד מנצח.
